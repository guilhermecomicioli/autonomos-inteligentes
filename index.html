O que são Autômatos Finitos?

Um autômato é um modelo matemático de um sistema com entradas, estados e saídas. Ele muda de um estado para outro dependendo dos estímulos (eventos) que recebe do ambiente.

Mapeamento Computacional no Contexto Agrícola:

  • Estados ($Q$): Representam as situações possíveis onde o maquinário ou sistema se encontra (ex: Sistema Desligado, Irrigando o Campo, Alerta de Evaporação).
  • Alfabeto de Entrada ($\Sigma$): Representa as leituras brutas vindas dos sensores de campo (ex: solo seco, chuva detectada, temperatura crítica).
  • Função de Transição ($\delta$): É a inteligência do sistema. São as regras lógicas que dizem: "Se o sistema está no estado X e o sensor ler Y, o sistema deve ir para o estado Z".

Vantagens dessa abordagem no campo:

Diferente de códigos cheios de condições confusas (vários if/else espalhados), um autômato garante que o sistema nunca entre em um estado inválido ou perigoso, evitando desperdício de água, quebra de maquinários ou falhas elétricas.
